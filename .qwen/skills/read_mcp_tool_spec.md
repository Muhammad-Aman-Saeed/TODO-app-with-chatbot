# Read MCP Tool Spec

## Description
Analyze a MCP tool definition from @specs/api/mcp-tools.md or document.

## Output Format
**Tool: [name]**
- Purpose: ...
- Parameters: ...
- Returns: ...
- Validation: user_id required, match authenticated
- Example call & response

## Usage
This skill helps parse and analyze MCP tool definitions from the specification documents to understand their purpose, parameters, return values, validation requirements, and example usage.
